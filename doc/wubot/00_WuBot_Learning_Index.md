## 🗓️ 推荐学习路线

### 第 1 天：建立全局（30 分钟）

```
① 读 01_Project_Overview.md        → 知道项目在干什么
② 打印 08_Quick_Reference.md       → 贴在显示器旁边
③ 跑一个 play.py 看效果            → 建立感性认识
```

### 第 2~3 天：掌握管线（训练期间阅读）

```
④ 读 02_Data_Pipeline.md           → 理解数据怎么来的
⑤ 读 03_Environment_Config.md      → 理解训练环境怎么搭的
⑥ 启动一个训练任务                  → 边训练边读
```

### 第 4~5 天：学会调参（看完训练结果后）

```
⑦ 读 04_Reward_Design.md           → 这是核心！理解 exp(−error/σ²)
⑧ 读 05_Tuning_Debugging.md        → 对照 TensorBoard 诊断问题
⑨ 调一次参数，重新训练              → 实践加深理解
```

### 第 6~7 天：规划后续

```
⑩ 读 06_Training_Roadmap.md        → 知道接下来训练什么
⑪ 读 07_Deployment_Guide.md        → 了解部署流程（等所有动作训完后用）
```
